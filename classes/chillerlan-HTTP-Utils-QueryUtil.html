By default, booleans will be left as-is (BOOLEANS_AS_BOOL) and may result in empty values. If $remove_empty is set to true, empty and null values will be removed from the array.

UTF-8 aware \parse_url() replacement.

public static parseUrl(string $url) : array<string|int, mixed>|null

The internal function produces broken output for non ASCII domain names (IDN) when used with locales other than "C".

On the other hand, cURL understands IDN correctly only when UTF-8 locale is configured ("C.UTF-8", "en_US.UTF-8", etc.).
